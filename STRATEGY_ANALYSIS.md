## Общая стратегия (обе стратегии совпадают по сути)

### 1. Фокус: OG-токены по нарративу

**Тезис:** Когда новый токен в какой-либо нарративной категории резко растёт,
**OG-токен в той же категории** копирует движение с задержкой 15–120 мин.

Примеры механики:
- Новый dog-мем вырастает x10 → ищем старейший dog-токен в этой нише → входим
- AI-нарратив снова взлетает в Twitter → находим OG AI-токен с историей → входим
- Новость о Solana/проекте → связанный старый токен реагирует

### 2. Алгоритм входа

```
1. Видим сильное движение (+50–200%) нового токена в категории
2. Ищем OG-токен той же категории (создан давно, есть история)
3. Покупаем зонд: $80–100
4. Если за 5–15 мин цена не падает — добавляем до нужного размера
5. НИКОГДА не усредняемся вниз — только добавляем к прибыльной позиции
```

### 3. Алгоритм выхода (ключевое отличие от большинства)

```
При первом TP (+30–50%):  продаём 30–40% позиции
При втором TP (+80–100%): продаём ещё 30–40%
Остаток (20–30%):         держим без стопа — это "лотерейный билет"
При ударе по лоссу (-5%): выходим ПОЛНОСТЬЮ, без раздумий
```

**Главный инсайт:** Убытки у обоих кошельков мизерные (-0.5% до -4%).
Winrate всего 52–56%, но средний выигрыш >> средний проигрыш.

### 4. Управление рисками

| Параметр | Значение |
|---|---|
| Размер позиции | $80–200 на новый токен |
| Максимальный риск на сделку | ~$5–10 (мгновенный кат) |
| Максимальная позиция | $400–500 только в проверенном движении |
| Одновременных позиций | ~5–15 (много параллельных, большинство — зонды) |
| Стоп-лосс | **Мгновенный**, не жмут ни секунды |
